Z-Shell Notes

What is Z-Shell:

Z-Shell is an extended version (superset) of Bash. It is the default shell for MacOS

Directory-Related Commands:

  1. pwd: prints current directory
  2. cd: changes current directory
  3. ls: lists files and directories in current directory
  4. ls -a: lists all files and directories in current directory
  5. rm [file-name]: removes file-name in current directory
  6. rm -rf [directory-name or file-name]: removes directory or file in current directory

Z-Shell Notes

What is Z-Shell:

Z-Shell is an superset of Bash. It is the default shell for MacOS

pwd

"pwd" prints the working directory

Flags:

  • pwd -L (default action)
  • pwd -P

cd

"cd [directory]" changes current working directory

Flags:

ls

"ls" lists stuff inside current working directory

Flags:

  • ls -l

    • Displays the mode, number of links, owner, group, size (in bytes), and time of last modification for each file.

Renaming Directories and Files:

Navigate to the parent directory of the directory you want to change and running
"mv [current-filename] [new-filename]"
